ATEC-Systems Javascript Framework

jATEC is a fast and concise JavaScript Library that simplifies HTML DOM handling and Ajax interactions for rapid web development. You are free to use the jATEC framework in your own projects.

No warranty no guarantee.
Use at your own risk.
